# Matthew Chrostowski — Portfolio Website

Personal portfolio site for Matthew Chrostowski, Senior BI Developer and Data Professional with 11 years of experience.

Live site: [mchrosto.github.io](https://mchrosto.github.io)

## About

A single-page portfolio showcasing professional experience, skills, and key achievements across data analytics, business intelligence, and financial analysis roles.

## Sections

- **Hero** — Summary and call to action
- **Skills** — SQL, Python, Tableau, Power BI, Excel, Leadership
- **Achievements** — Key quantified impact metrics
- **Experience** — Full career timeline
- **Education** — B.S. Accounting, SUNY Oswego
- **Contact** — Email and LinkedIn

## Tech Stack

- HTML5
- CSS3 (no frameworks)
- Vanilla JavaScript

## Local Development

No build tools required. Open `index.html` directly in a browser.

## Deployment

Hosted on GitHub Pages. Any push to `main` updates the live site automatically.
